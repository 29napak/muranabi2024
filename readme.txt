
ーーーーデータ編集ーーーーー
1.Terminalを開いて
2. npm run dev
3.http://localhost:5173/ でページ表示確認
4.ファイル編集　変種したらセーブしてページ表示確認
5.問題なかったら　crlt+c　でページ表示閉じる
6.npm run build
7.distフォルダを開いてindex.htmlをダブルクリックしてページ表示確認
8.ページ表示問題なかったらデザインサーバーにindex.htmlをアップする

ーーーーteamsiteにアップーーーーー
1.distフォルダの中にあるindex.htmlファイルをpc.htmlに名前変更
2.teamsiteにアップ
